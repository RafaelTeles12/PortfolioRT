Sobre Mim

Olá, me chamo Rafael Teles, sou estudante de Ciências da Computação e apaixonado por programação e hardware. Como profissional, sou colaborativo, paciente, empático, calmo, competitivo, determinado, organizado e amigável. Desejo desenvolver minha carreira sob a mentoria e estímulo de uma empresa relacionada à tecnologia da informação.

Formação

Possuo formação internacional, tendo concluído o ensino médio em Bruxelas, Bélgica. Atualmente, resido em São Paulo, onde estou cursando Bacharelado em Ciência da Computação na Universidade Nove de Julho, com previsão de conclusão em julho de 2025.

Projetos

  • RetroBot

    O RetroBot é um projeto acadêmico, desenvolvido no formato de uma aplicação web. Nessa iniciativa, desenvolvi um chatbot com foco na estética retrofuturista utilizando as linguagens HTML, CCS e Javascript. Para o uso do chatbot, foram integradas duas as APIs Chat GPT 3.5 e Google Cloud Vision.

  • API NB

    O projeto API NB se trata de uma API desenvolvida em Python, utilizando a bibliotéca Flask e o modelo de Naive Bayes com o objetivo de realizar a classificação de curtas avaliações. Nessa iniciativa, foram exploradas as disciplinas de aprendizado de máquina e processamento de linguagem natural.
